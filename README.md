# 🚀 End-to-End CI/CD Pipeline using Jenkins, Docker & AWS

## 📌 Project Overview
This project demonstrates a complete DevOps workflow where code is automatically built, tested, and deployed using CI/CD pipelines.

## ⚙️ Tech Stack
- Jenkins (CI/CD Automation)
- Docker (Containerization)
- AWS EC2 (Cloud Deployment)
- Git & GitHub (Version Control)

## 🔧 Features
- Automated build and deployment pipeline
- Dockerized application for consistent environment
- Continuous integration using Jenkins
- Deployment on AWS EC2 instance

## 🚀 Workflow
1. Developer pushes code to GitHub
2. Jenkins triggers build automatically
3. Docker image is created
4. Application deployed on AWS EC2

## 🎯 Outcome
- Reduced manual deployment effort
- Faster and reliable deployments
- Improved development workflow
